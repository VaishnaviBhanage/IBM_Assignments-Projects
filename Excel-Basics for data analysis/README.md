## Montgomery Fleet Equipment Inventory Analysis

### Course
IBM Data Analyst Professional Certificate

### Type
Course-based Excel assignment

### Description
This assignment focuses on analyzing the Montgomery County fleet equipment inventory dataset using Microsoft Excel. The work was completed as part of a guided course assignment and is divided into two parts.

### Files Included
- **Montgomery_Fleet_Equipment_Inventory_FA_PART_1_END.xlsx**  
  Data cleaning, formatting, and basic analysis of fleet equipment inventory.

- **Montgomery_Fleet_Equipment_Inventory_FA_PART_2_END.xlsx**  
  Advanced analysis using Excel features such as filters, sorting, and summary calculations.

- Basic data analysis in Excel
