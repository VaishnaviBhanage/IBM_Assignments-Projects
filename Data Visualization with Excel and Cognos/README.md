## Car Sales Analysis & Dashboard

### Course
IBM Data Analyst Professional Certificate

### Type
Course-based data visualization assignment

### Description
This assignment focuses on analyzing car sales and profit data by model and presenting key business insights through a dashboard created using Looker Studio. The work was completed as part of a guided course assignment to practice end-to-end data analysis and visualization.

### Files Included
- **CarSalesByModelEnd.xlsx**  
  Contains the cleaned and prepared dataset used for analysis, including quantity sold, profit, dealer information, and model-wise breakdowns.

- **Sales_Dashboard.pdf**  
  PDF export of a Looker Studio dashboard visualizing key sales and service insights such as:
  - Quantity sold by car model
  - Profit by dealer
  - Monthly sales and profit trends
  - Sales vs service comparison
  - Units affected by different service categories

### Key Insights Covered
- Comparison of sales performance across car models
- Identification of high- and low-performing dealers based on profit
- Monthly trends in quantity sold and profit
- Relationship between sales and service metrics

### Skills Practiced
- Data preparation for visualization
- Dashboard creation using Looker Studio
- Data visualization and storytelling
- Interpreting business metrics (sales, profit, units)

### Tools Used
- Microsoft Excel (data preparation)
- Looker Studio (dashboard creation)

### Notes
This is a structured, course-based assignment completed to gain hands-on experience with data analysis and visualization workflows.
